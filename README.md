## 📌 Deskripsi Proyek
Proyek ini mengimplementasikan **Algoritma Genetika (Genetic Algorithm)** untuk menyelesaikan masalah optimasi fungsi matematika:

$$f(x) = -x^2 + 4x + 10$$

Tujuannya adalah mencari nilai $x$ pada rentang $[-10, 10]$ yang menghasilkan nilai *fitness* maksimum.
